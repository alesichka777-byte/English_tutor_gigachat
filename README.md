# English Tutor Telegram Bot

Телеграм-бот-репетитор по английскому языку на основе GigaChat.

## Возможности

- Запуск по команде `/start`.
- Выдаёт задания по английскому (лексика/грамматика/чтение).
- Даёт обратную связь и объяснения на основе ответов пользователя.

## Технологии

- Python
- GigaChat API
- aiogram
- python-dotenv

## Как запустить

1. Клонировать репозиторий или скачать код.
2. Установить зависимости:
   `pip install -r requirements.txt`
3. Создать файл `.env` и добавить туда токены:
   - `GIGACHAT_API_KEY=...`
   - `TELEGRAM_BOT_TOKEN=...`
4. Запустить бота:
   `python main_gigachat.py`
